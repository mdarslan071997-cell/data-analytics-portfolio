# 📊 Sales Data Analysis Project

## 📌 Objective
Analyze sales data to identify revenue trends, top-performing products, and business insights.

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## 🔍 Steps Performed
1. Data Cleaning
2. Handling Missing Values
3. Exploratory Data Analysis (EDA)
4. Revenue Analysis
5. Monthly Sales Trend
6. Business Insights

## 📈 Business Questions
- Which product generates highest revenue?
- What is monthly sales growth?
- Which category performs best?
